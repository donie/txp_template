Installation

The installation is pretty straightforward, assuming you already have a working Textpattern installation (4.0.5+ required) and an FTP access to it's directory. Just follow the steps below.

  1. Upload static content

    Upload “gaia” directory of the template package with it's contents to the root of your Textpattern installation. It contains all the styles, images and scripts the template needs.

  2. Set required preferences

    Go to admin -> preferences -> basic page in your Textpattern admin interface.

    Set “Automatically append comments to articles?” to “No”. “Present comments as a numbered list?” should be set to “Yes”.

  3. Set up pages

    Every text file in the "pages" directory of the package corresponds to a template page with the same name. Go to presentation -> pages, then create (or replace) all appropriate pages using the contents of that directory.

  4. Set up forms

    Every text file in the "form" directory of the package corresponds to a form with the same name. Go to presentation -> forms, then create (or replace) all appropriate forms with the following types:

    Form name Form type
    archive_article article
    archive_heading misc
    footer misc
    full_article article
    header misc
    search_results article
    static_article article
    (existant forms) (default)

    Important note: some text editors/viewers don't display header form content properly for some reason. If this happens, try a different editor.

  5. Set up sections

    Go to presentation -> sections, create “article” section and do the following:

    • For the about section, set “Uses page” to “static_page”
    • For the article section, set “Uses page” to “default”
    • Create a section with name “archive”. Set “Uses page” to “archive”. Set other options for it to “No”.

    Don't forget to save after every action.

  6. Install plugins

    Go to admin -> plugins, then install plugins by copying the content of each file in the “plugins” directory into an appropriate field, clicking "install", and then activating it by clicking “No“ under the “Active” column in the plugin list.

You're done, hooray! View the website and enjoy.

If you have encountered any problems with the installation, feel free to ask for help in the Gaia TXP forum topic.
